# Basket Craft — Merchandising Dashboard

**Live app:** https://basket-craft-dashboard-7ot2szphhenscxqjbckgv2.streamlit.app

A Streamlit dashboard built for Maya, Head of Merchandising at Basket Craft. Connects to Snowflake to answer two key questions:

- Which products drove the most revenue over a given period?
- Which products get bought together most often?

## Features

- **KPI cards** — Total revenue, orders, AOV, and items sold with month-over-month deltas
- **Revenue trend** — Daily revenue line chart with date range filter
- **Top products** — Horizontal bar chart of products ranked by revenue
- **Bundle finder** — Co-purchase table showing what gets bought with any selected product, with CSV download
